# Fog 4-17 / 4-18 Design

## Goal

为浓雾世界补上 `4-17` 与 `4-18` 两关。

- `4-17`：无浓雾的后院传送带特殊关，传送带只产出 `brine_pot`，并让全游戏已实现的非 Boss 僵尸都有机会出现。
- `4-18`：浓雾世界终章 Boss 关，仍为传送带模式，传送带包含第四世界全部植物但不包含 `lily_pad`，并新增一个雾系终章 Boss。

## Approach

### 方案 A：只补关卡数据，不新增 Boss

直接把 `4-18` 做成高压普通关，复用现有非 Boss 僵尸。

- 优点：改动最小，回归面小。
- 缺点：不符合“Boss 关卡”的预期，第四世界缺少明确终章。

### 方案 B：复用旧 Boss，换一个世界皮肤

让 `4-18` 直接复用 `night_boss` 或 `pool_boss`，只调整关卡数据和植物池。

- 优点：实现快，稳定性高。
- 缺点：世界辨识度差，终章显得像复用而不是完成版。

### 方案 C：新增轻量 Fog Boss

新增一个矢量绘制的 `fog_boss`，复用现有 Boss 通用框架，只补雾系召援、盐沼封锁和压前场三类技能。

- 优点：满足 Boss 终章感，同时改动仍可控。
- 缺点：需要补运行时逻辑、图鉴和测试。

## Recommendation

采用方案 C。

原因：

- 现有代码已经有完整的 Boss 生命周期、血条、阶段切换、召援和技能循环框架。
- 新增一个轻量 `fog_boss` 的成本明显低于再做一套特殊系统。
- 这样 `4-18` 会和 `1-17 / 1-18 / 3-18` 的终章结构对齐。

## Level Design

### 4-17

- 世界：`fog`
- 地形：新增无雾后院池塘地形，保留后院泳池外观和水路，但不启用雾遮挡。
- 模式：`conveyor`
- 传送带：`conveyor_plants` 全部为 `brine_pot`
- 僵尸：覆盖当前游戏中所有非 Boss 僵尸；为避免支持型 Boss 单位混入，不包含任何 `boss: true` 项。
- 定位：利用盐沼减速与格位控场去拖住全阵营混编。

### 4-18

- 世界：`fog`
- 地形：标准浓雾后院
- 模式：`conveyor`
- 标记：`boss_level: true`
- 传送带：覆盖第四世界植物全集，但显式排除 `lily_pad`
- Boss：新增 `fog_boss`
- Boss 技能：
  - 召援：从浓雾世界与后院体系僵尸池中召出混编援军
  - 盐沼封锁：在前场生成持续减速的 `bog_pool`
  - 迷雾压制：对各行前排植物造成压力伤害，并伴随雾系特效

## Runtime Changes

- 新增一个 `clear_backyard` 地形，使第四世界可以使用后院池塘地形但不触发雾系统。
- 新增 `fog_boss` 定义、图鉴顺序、图鉴文案、技能循环、阶段切换、右侧持续召援和独立绘制。
- 更新世界元数据，把浓雾世界范围改为 `4-1 ~ 4-18`。
