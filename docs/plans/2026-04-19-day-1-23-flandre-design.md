# 白天世界 1-23 芙兰朵露 Boss 关设计

## 目标

新增白天世界支线终章 `1-23`，接在 `1-22` 之后。关卡继续放在白天世界最右侧，需要横向滑动才能看到；流程为传送带 Boss 关，半程由帕秋莉作为道中 Boss 锁住进度，终幕由芙兰朵露登场完成收尾。

## 关卡结构

- 世界归属：`day`
- 关卡编号：`1-23`
- 解锁条件：必须先完成 `1-22`
- 关卡类型：`conveyor`
- 波次：`9` 个大波节点，其中半程锁进度，终幕为最终 Boss 波
- BGM 切换：
  - 道中：`/Users/hecrereed/Downloads/1-23道中.mp3`
  - 终幕：`/Users/hecrereed/Downloads/1-23终末.mp3`

## 场景主题

地形采用新的“血月玩具屋顶”风格，不复用此前血图书馆、时钟厅或血月庭院背景。视觉关键词：

- 红月与深酒红色天幕
- 屋顶 / 阁楼 / 玩偶房混合场景
- 布偶、丝带、积木、玻璃晶翼感装饰
- 部分地皮为普通落点，部分地皮为屋顶瓦片，需要 `flower_pot`

实现上仍复用现有战斗系统，不切独立场景：

- 新增 `terrain` 标识与对应背景/地板绘制
- 新增一组按列或固定图案分布的“需花盆格”判定
- 传送带植物中加入足量 `flower_pot`，但不让整局退化成纯花盆局

## 植物池

传送带遵循“能打终盘高压 Boss，但不塞无意义功能牌”的原则：

- 保留高强度输出、控制、灰烬、投射、屋顶适配植物
- 补入 `flower_pot`
- 不放 `grave_buster`、纯经济型植物、明显无效的场景专用废牌
- 兼顾单点爆发、整行压制、抗压前排和清杂能力

## 道中 Boss：帕秋莉

帕秋莉继续作为 `patchouli_boss`，但在 `1-23` 的道中阶段升级为更夸张的属性魔法主导：

- 锁进度点出现在约 `50%`
- 需要击败帕秋莉后，后半程推进恢复
- 保持右侧杂兵持续入场
- 在现有火 / 水 / 木 / 金 / Royal Flare 基础上，扩展到更完整的元素轮转和更华丽的法阵特效

技能方向按东方设定靠近七曜与多属性魔法：

- 火符 / 水符 / 木符 / 金符 / 土符
- 日月系强化符卡
- 贤者之石式全屏压制
- 书页 / 使魔 / 法阵召援

## 终幕 Boss：芙兰朵露

新增 `flandre_boss`，使用用户提供的 8 格图集，先离线裁剪、去白边，再保存到素材目录作为预烘焙左向帧，不在运行时做横向翻转。

Boss 设计约束：

- 右侧悬停 Boss，不向房子推进
- 使用现有 hover boss 框架：悬停换行、技能循环、阶段切换、底部五段血条、持续增援
- 技能与表现突出“玩具、破坏、血月、禁忌弹幕”
- 至少 `11` 个技能，特效与已有露米娅 / 咲夜 / 蕾米莉亚明显区分

建议技能池：

1. `Laevatein` 直线高热切割
2. `Four of a Kind` 分身压场
3. `Kagome, Kagome` 环形封锁
4. `Starbow Break` 彩弹散射
5. `禁忌玩偶箱` 玩偶召援与落地爆裂
6. `红月晶刺` 多点穿刺
7. `破坏开关` 指定高血量植物爆破
8. `血玩具风暴` 玩偶与碎片旋风
9. `Secret Barrage` 扇面超密弹
10. `玩具审判庭` 多列格位轰击
11. `Cranberry Trap` 地面猩红陷阱或玩偶圈

其中分身、禁忌玩偶、红月晶刺和血玩具风暴用于建立本关独特识别度。

## 运行时实现策略

采用方案 A，完全复用现有东方支线 Boss 总框架，不新建独立战斗模式：

- `level_defs_day.gd` 中新增 `1-23`
- `world_data.gd` 扩展白天世界描述与范围
- `zombie_defs.gd` 新增 `flandre_boss`
- `normalize_boss_frames.gd` 新增 `flandre_boss` 图集裁剪入口
- `game.gd` 中补：
  - `1-23` 的中 Boss 锁进度流转
  - 血月玩具屋顶背景和需花盆格规则
  - 芙兰朵露技能循环、阶段切换、增援、绘制与特效
  - 道中帕秋莉在 `1-23` 的增强分支表现

## 测试重点

- `1-23` 存在、接在 `1-22` 后、锁定条件正确、位于白天世界最右侧
- `1-23` 为传送带 Boss 关，传送带包含 `flower_pot`

- 道中 `patchouli_boss` 会锁半程进度
