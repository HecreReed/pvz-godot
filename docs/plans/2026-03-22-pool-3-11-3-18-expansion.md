# Pool 3-11 To 3-18 Expansion Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add pool world levels `3-11` through `3-18`, new pool-era plants and zombies, and fix mismatched attack/effect behaviors and missing Plant Food coverage in the existing roster.

**Architecture:** Reuse the current single-script gameplay architecture in `scripts/game.gd` and `scripts/game_defs.gd`, extending the existing plant/zombie dictionaries, projectile/effect system, and level event pipeline instead of refactoring modules first. Add a small number of generic helpers for custom projectile motion, stealth reveal, shield-break residue, crush movement, and special row/terrain spawning so the new content stays testable without destabilizing the current campaign.

**Tech Stack:** Godot 4 GDScript, dictionary-driven gameplay definitions, headless SceneTree regression tests.
